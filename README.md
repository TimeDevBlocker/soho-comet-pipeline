# soho-comet-pipeline
Automated SOHO LASCO C2 and C3 comet search

***
<br/>

## Features and functions
1. Batch pre-processing
2. Comet detection
3. Sends possible comet candidate to your email address
<br/>

## How to use

```python
pip install -r requirements.txt

python main.py
```
